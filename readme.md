# Yukigram PKGBUILD

Up-to-date pkgbuild for [Yukigram](https://github.com/Revincx/yukigram) for Arch-based linux distros.

## Usage

```shell
# Clone this repo and go inside
git clone https://github.com/RuaGram/PKGBUILD yukigram-pkgbuild
cd yukigram-pkgbuild

# Build and install (-s: install required deps, -i: also install)
makepkg -si
```
